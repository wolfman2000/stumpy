Using Stumpy

General Usage

The mouse is heavily used for this tracker. Use the mouse to mark which items you have, which bosses you have defeated, and which overworld locations have been cleaned out.

Settings

If the settings are confusing, the explanations are here.

Mode

This states the general mode of the game.

  • Standard The standard opening of the game plays out. One sword is guaranteed.
  • Open The rainy intro is pre-done. You can start from Sanctuary. No sword is guaranteed.
  • Swordless Open Mode, but without any swords at all! The hammer can be used against Ganon, at least.
Goal

What must be done to finish the game?

  • Ganon Get the seven crystals, finish off Agahnim in Ganon's Tower, then finish the big pig.
  • All Dungeons All dungeons must be beaten. Agahnim must be killed in both Castle Tower and Ganon's Tower.
Difficulty

How hard is the seed?

  • Easy Double the chances for progressive items. Quarter magic available.
  • Normal The default selection.
  • Hard Many safety items are removed.
  • Expert Heart containers are removed.
  • Insane Heart pieces are removed.
Shuffling Mode

What is shuffled?

  • Normal The normal items are shuffled.
  • Keysanity Keys, maps, and dungeons are also shuffled.
Go Mode Label

Show an indicator if you can just rush to the end?

  • No No special label shows up when the end is near.
  • Yes The Go Mode label shows. It includes visual indication of potential sequence breaks.
Enemy Shuffle Option

Are enemies (well, bosses) shuffled? This affects the logic.

  • Normal The normal enemies and bosses are at the dungeons.
  • Boss The bosses are shuffled. Are dungeons no longer beatable on the first try? Left/right click on the boss images to change the boss.
